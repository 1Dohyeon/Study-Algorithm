기본적으로 문자열 S에 원하는 문자열 A가 있는지 탐색하여 찾아내는 알고리즘.

### 수도 코드

```
naiveMatching(A, P) 
{ 
	▷ n : 배열 A[ ]의 길이, m : 배열 P[ ]의 길이 
	for i ← 1 to n- m+1{ 
		if (P[1… m] = A[ i… i+ m-1]) 
		then A[i] 자리에서 매칭이 발견되었음을 알린다; 
	} 
}
```
- A[1...n] : 텍스트 문자열, P[1...m] : 패턴 문자열 (n>m)
- 텍스트 문자열 A[1...n]이 패턴 문자열 P[1...m]을 포함하는지 알아보는 메서드이다.
- 기본적으로 전부 탐색하는 방법으로 시간복잡도는 **O(mn)** 이다.

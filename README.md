# Study-Algorithms

### Folder
> 여러 알고리즘 개념을 정리해서 폴더명 대로 분류 해놓음.

- DynamicProgramming : 동적프로그래밍 알고리즘
  - 최장 공통 부분 순서(LCS) 알고리즘
  
- Graph : 그래프 알고리즘 정리 
  - 탐색
    - BFS
    - DFS
    
  - 최소신장트리
    - Prim 알고리즘
    - Kruskal 알고리즘
    
  - 위상정렬
    - TopologicalSort 알고리즘
    
  - 최단경로
    - Dijkstra 알고리즘
    - Bellman-Ford 알고리즘
    - Floyd-Warshall 알고리즘
    
  - 싸이클이 없는 그래프에서의 최단 경로
    - DAT
    
  - 강연결요소
    - StronglyConnectedComponent 알고리즘
      
- Greedy : 눈 앞에 보이는 최적의 해를 구하는 알고리즘
  - 그래프 최소신장트리를 구하는 Prim 알고리즘
  - Huffman Code 알고리즘
  
- Selection : 선택 알고리즘 정리
  - 퀵 정렬 방식을 이용한 선택 알고리즘
  - 중앙값의 중앙값을 이용한 선택 알고리즘
  
- Sort : 정렬 알고리즘 정리
  - 버블정렬
  - 삽입정렬
  - 선택정렬
  - 퀵정렬
  - 병합정렬
  - 힙정렬
  - 계수정렬
  - 기수정렬
  
- StringMatching : 문자열 매칭 알고리즘 정리
  - 원시매칭
  - Automata
  - KMP
  - Rabin-Karp
  
- Tree : 트리 알고리즘 정리
  - 2-3트리 
  - B트리 
  - 레드블랙트리
